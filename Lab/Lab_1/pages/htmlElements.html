Elements

HTML uses elements to describe the structure of the pages. All structured elements have a opening and closing tag. The tags acts like containers. Describing the type of information that lies in between.

Head
Contains information about the page
Title
The contents are shoen in the top of the browser.
Body
Indicates that all that is included inbetween will be shown in the browser.
Header
Used to contain the header content of a site. It normally contains other tags as the navigation tag, to move in between other internal sites.
Footer
Contains the footer content of a site. Normally it contains the address, the author and copy right
Nav
Contains the navigation menu, or other navigation functionality for the page.
Article
Contains a standalone piece of content that would make sense if syndicated as an RSS item, for example a news item.
Section
Used to either group different articles into different purposes or subjects, or to define the different sections of a single article.
Aside
Defines a block of content that is related to the main content around it, but not central to the flow of it.
Heading
H1 contains the main heading and the other subsequent h2, h3, etc are sub-headings
Paragraph
Contains a paragraph, giving a description or a defenition of the heading.
